# How to Update the Website

Open a terminal, navigate to this folder, and run:

```bash
./start_gui.sh
```

Then open **http://localhost:5001** in your browser.

From there you can:
- **Profile** — update your bio, interests, education, social links
- **Publications** — add, edit, or remove papers
- **News & Events** — add conference posters, talks, awards
- **CV / Resume** — upload a new PDF
- **Preview & Publish** — preview locally, then commit and push to make changes live

Changes go live at **https://ramcha24.github.io** within ~1 minute of publishing.
